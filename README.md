# CourtLink – Web-Based Legal Assistance Platform

CourtLink is a web-based legal assistance platform developed as a team project to simplify legal service management through a centralized and user-friendly system. It enables clients to file cases, manage legal documents, connect with advocates, and utilize AI-powered legal assistance.

## Features
- Secure user authentication and role-based access
- Case filing and management
- Advocate assignment and consultation
- Legal document upload and management
- AI Legal Assistant
- AI Document Summarizer
- Online payment integration using Razorpay
- Responsive user interface

## Tech Stack
### Frontend
- React.js
- Vite
- Tailwind CSS
### Backend
- Node.js
- Express.js
### Database
- PostgreSQL
### Tools & Services
- JWT Authentication
- Razorpay
- Cloudinary
- Nodemailer
- Google reCAPTCHA
- Groq AI API

## Project Goal
The goal of CourtLink is to provide a secure and accessible digital platform that simplifies legal service management and improves communication between clients and advocates through modern web technologies and AI-powered features.
