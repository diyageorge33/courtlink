CourtLink is a web-based legal service platform designed to simplify case management, advocate interaction, and document handling through a centralized system.
The platform improves accessibility and efficiency by integrating legal services into a single user-friendly application.

Features
User authentication and role-based access
Case management system
Advocate assignment
Document upload and handling
Notifications and case updates
AI-based assistance
Responsive user interface

Tech Stack
Frontend
React.js
Vite
Tailwind CSS

Backend
Node.js
Express.js

Database
PostgreSQL

Goal
The main goal of CourtLink is to provide an efficient and accessible digital platform that simplifies legal service management for clients, advocates, and administrators.
